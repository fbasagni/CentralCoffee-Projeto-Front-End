# CentralCoffee – Projeto Front-End  
Atividade Somativa • Fundamentos de Programação Web • PUCPR  
Autora: **Francine Basagni**

---

## 1. Sobre o Projeto

O *CentralCoffee* é um pequeno site desenvolvido para praticar os fundamentos de **HTML**, **CSS** e **JavaScript**, aplicando na prática tudo o que foi estudado nas primeiras semanas da disciplina.  

A ideia foi criar um site simples, funcional e organizado, com navegação clara entre páginas e um formulário capaz de capturar dados via **método GET**, conforme solicitado na atividade.

O projeto simula uma cafeteria inspirada no estilo acolhedor de seriados, com foco em estrutura, semântica e boas práticas de front-end.

---

## 2. Objetivos Desenvolvidos

Este projeto foi construído para demonstrar:

- Estruturação correta de páginas usando **HTML semântico**.  
- Organização visual utilizando **CSS**, com identidade consistente entre as páginas.  
- Navegação funcional entre arquivos através de um menu fixo presente em todas as telas.  
- Manipulação de parâmetros enviados via **GET**.  
- Separação e organização dos arquivos em diretórios (padrão recomendado):  
  - `/img`  
  - `/css`  
  - `/js`  

Cada etapa foi implementada de forma simples, mas limpa e coerente — focando na clareza e na correta aplicação dos conceitos fundamentais da disciplina.

```plaintext

├── index.html
```

## 4. Tecnologias Utilizadas

  - HTML5
  - CSS3
  - JavaScript (básico)

## 5. Como Visualizar o Projeto

  - Baixe ou clone o repositório.
  - Abra o arquivo index.html diretamente no navegador.
  - Utilize o menu para navegar entre as páginas.
  - Preencha o formulário em form.html — os dados enviados via GET serão exibidos em formAction.html.

## 6. Pontos Entregues

  - Página inicial (index.html) estruturada.
  - Página de formulário (form.html) com envio GET.
  - Página de recebimento (formAction.html) exibindo os dados enviados.
  - Página informativa (about.html).
  - Navegação consistente entre todas as páginas.
  -  Organização em pastas (img, css, js).
  -  Aplicação dos fundamentos de HTML, CSS e JavaScript conforme solicitado.
